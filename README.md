# MicroAvgCalc
This MicroAvgCalc calculates the average of a rolling window of numbers fetched from a Test Server API. It provides a REST API endpoint /numbers/{numberid} supporting:

- p: Prime numbers
- f: Fibonacci numbers
- e: Even numbers
- r: Random numbers
